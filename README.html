Blockchain Charity Donation System

Nền tảng từ thiện phi tập trung giúp minh bạch, tin cậy và trực quan hóa quá trình quyên góp trên blockchain Ethereum.

🎯 Mục tiêu

Hệ thống hỗ trợ quyên góp minh bạch bằng cách ghi nhận mọi giao dịch lên blockchain, giúp người dùng và nhà tài trợ theo dõi nguồn tiền rõ ràng và giảm rủi ro gian lận.

  • Minh bạch: Lịch sử quyên góp và phân phối được lưu trên blockchain.
  • Tin cậy: Hợp đồng thông minh kiểm soát chặt chẽ luồng quỹ.
  • Dễ sử dụng: Giao diện web thân thiện cho cả người quyên góp và tổ chức.

✨ Tính năng chính

  • Đăng ký và xác thực người dùng
  • Quyên góp trực tiếp bằng tiền mã hóa
  • Theo dõi chiến dịch, số tiền đã nhận và trạng thái hỗ trợ
  • Chứng thực minh bạch bằng khối chuỗi
  • Hỗ trợ hiển thị đồ họa trong dashboard và báo cáo lịch sử

🧱 Cấu trúc dự án

  • app/ — Backend Python cho API và xử lý logic
  • frontend/ — Giao diện web React
  • firmware/ — Mã Arduino / ESP8266 nếu có phần tương tác phần cứng
  • requirements.txt — Thư viện Python cần cài
  • package-lock.json — Khóa dependency cho frontend

🚀 Hướng dẫn nhanh

Để chạy ứng dụng, bạn có thể làm theo các bước cơ bản sau:

  • Tạo và kích hoạt virtualenv cho Python
  • Cài đặt dependencies với pip install -r requirements.txt
  • Khởi động frontend nếu có npm
  • Chạy backend từ app/main.py

Nếu cần, tôi có thể viết thêm phần hướng dẫn chi tiết cho cả backend và frontend.
